# Conway's Game of Life

## Running

```bash
cargo run --release
```

## Controls

- Mouse: Left click toggles cells, dragging draws lines.
- <kbd>P</kbd>: Toggle pause.
- <kbd>space</kbd>: Frame step (enables pause if not already paused)
- <kbd>R</kbd>: Randomize
- <kbd>F</kbd>: Make 100 updates per frame
- <kbd>escape</kbd>: Quit
